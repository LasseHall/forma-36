Notifications give immediate feedback about an action triggered or experienced by an user. They will dismiss after 6 seconds, or after being clicked.

```jsx
<Button
  buttonType="muted"
  onClick={() => Notification.success('Your preferences have been updated')}
>
  Show notification
</Button>
```

## Best practices <a name="best-practices" />

- Consider where the notification will be placed and which variation is most appropriate 
- Don't confuse notifications with notes, which persist in the UI and do not dismiss

## Copy considerations <a name="copy-considerations" />

- Use clear and succinct copy, since the notification will only be available for 6 seconds
- Use active voice, present tense, and consider tone of voice depending on the circumstance
- Use sentence style caps (in which only the first word of a sentence or term is capitalized)

## Variations <a name="variations" />

### Customized notifications <a name="customized" />

```jsx
<React.Fragment>
  <Button
    buttonType="muted"
    onClick={() => Notification.success('This is a success message')}
  >
    Show success message 
  </Button>
  <Button
    buttonType="muted"
    onClick={() => Notification.error('This is an error message')}
  >
    Show error message
  </Button>
  <Button
    buttonType="muted"
    onClick={() => Notification.warning('This is a warning message')}
  >
    Show warning message
  </Button>
</React.Fragment>
```

### Positioned notifications <a name="positioned" />

```jsx
<React.Fragment>
  <Button
    buttonType="muted"
    onClick={() => {
      Notification.setPosition('top');
      Notification.success('Notification message on top');
    }}
  >
    Show on top
  </Button>
  <Button
    buttonType="muted"
    onClick={() => {
      Notification.setPosition('bottom');
      Notification.success('Notification message on the bottom');
    }}
  >
    Show on the bottom
  </Button>
  
</React.Fragment>
```

### Long notifications <a name="message" />

```jsx
<React.Fragment>
  <Button
    buttonType="muted"
    onClick={() => {
      Notification.success('Long notifications will likely span into multiple lines, which will improve readibility');
    }}
  >
    Show long message
  </Button>
  
</React.Fragment>
```
